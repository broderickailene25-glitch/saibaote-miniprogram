# 赛宝特水墨画素材库

## 设计风格

三图融合：工笔农作物 + 土壤科普 + 写意点缀

- 图1（茶枝）：极简水墨工笔，红果绿叶点缀，大量留白
- 图2（腐植酸土壤剖面）：科普级农业插画，绿色系，展示「产品做了什么」
- 图3（莲蓬）：极简写意，一朵一梗，灰色水墨 + 一个小红点

## 中国传统色谱

| 颜色名 | 色值 | 用途 |
|--------|------|------|
| 黛绿 | #4A7C59 | 深沉的绿，比 #2E7D32 高级 |
| 竹青 | #7B8B6F | 竹子的青灰色 |
| 缃叶 | #ECC57C | 嫩叶黄，代表生长 |
| 朱砂红 | #C23B22 | 印章、点缀 |
| 墨色 | #3B3B3B | 不是纯黑 |
| 月白 | #D6ECF0 | 淡蓝白 |
| 素色 | #F5F0E8 | 宣纸底色 |
